# AutoRent - Аренда автомобилей

Веб-приложение для аренды автомобилей с современным интерфейсом.

## Технологии

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- React Router

## Запуск проекта

### Установка зависимостей

```bash
npm install
```

### Режим разработки

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:8080

### Сборка для продакшена

```bash
npm run build
```

### Предпросмотр сборки

```bash
npm run preview
```

## Структура проекта

```
src/
├── assets/        # Изображения и статические файлы
├── components/    # React компоненты
│   └── ui/        # UI компоненты (shadcn/ui)
├── data/          # Моковые данные
├── hooks/         # Кастомные хуки
├── lib/           # Утилиты
└── pages/         # Страницы приложения
```
